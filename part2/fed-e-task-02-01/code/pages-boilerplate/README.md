# 文档说明
>gulp是任务执行器，内部功能在gulpfile.js文件中

暴露命令：
1. 清除打包文件夹命令
>yarn gulp clean

2. 运行项目命令
>yarn gulp develop

3. 打包项目命令
>yarn gulp build
*****
